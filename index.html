Ethics and Persuasion

The control allows you to highlight, and adjust the ethical level of, each element of persuasion shown. Should you disagree with the ethical level shown for any specific persuasive element, check I disagree with this rating and select the ethical level you believe that persuasive element is demonstrating. To hide the ethical controls, simply click the control.

Use the controls below to select which types of persuasion are shown when the control is used. Simply select which types you would like highlighted. You can also reset the ethical levels of all types of persuasion shown by using the master reset control. To close this sidebar, click the control.
